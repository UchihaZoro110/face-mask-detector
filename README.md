# Face Mask Detector 😷

This is a Streamlit web app that predicts whether a person is wearing a mask or not, based on an uploaded image. The model was trained on a Kaggle dataset using PCA for dimensionality reduction combined with a Machine Learning classifier.

## Features
- Real-time prediction from uploaded images
- PCA-based dimensionality reduction
- Trained ML model (scikit-learn)

## Tech Stack
- Python
- Streamlit
- OpenCV
- scikit-learn
- joblib

## Model Files
`final_model.pkl` is included in this repo.  
`final_pca.pkl` is automatically downloaded from Google Drive on first run (via `gdown`).
